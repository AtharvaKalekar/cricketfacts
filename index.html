Cricket Facts Generator

A simple, single-page web application that displays random cricket facts. The application includes 50 interesting facts about cricket history, records, and terminology.

Features

  • Random fact generation with a single click
  • 50 unique cricket facts
  • Fact number tracking
  • Clean, responsive design
  • Mobile-friendly interface

Technologies Used

  • HTML5
  • CSS3
  • JavaScript (Vanilla)

Usage

  1. Download the index.html file
  2. Open index.html in any modern web browser
  3. Click the "Generate Fact" button to display a new random cricket fact
  4. The fact number will be displayed below each fact

Structure

The application is contained in a single index.html file that includes:

  • HTML structure
  • CSS styling in the <style> tag
  • JavaScript code in the <script> tag
  • JSON data containing 50 cricket facts

Preview

When loaded, the application displays:

  • A header titled "Cricket Facts Generator"
  • A fact display box with the current fact
  • A fact number indicator
  • A "Generate Fact" button

Browser Compatibility

The application works in all modern browsers including:

  • Chrome
  • Firefox
  • Safari
  • Edge
